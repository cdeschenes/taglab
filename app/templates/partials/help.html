Quick reference for TagLab features

cdeschenes/taglab

Library Explorer

  • Browse your library by artist and album in the left sidebar
  • Type in the filter box to narrow results by artist or album name
  • Sort A–Z or by Recently Added using the sort bar above the explorer
  • Click an album to open the tag editor in the main panel
    • {% if navidrome_enabled %}
  • Filter by Navidrome favorites (♥) or minimum star rating using the Navidrome dropdowns
    • {% endif %}

Tag Editing

  • Edit shared album fields (Album Artist, Year, Genre, Label, Country) at the top of the album editor
  • Click Apply to All to bulk-copy a shared field value to every track
  • Click any track row to open the per-track editor with the full tag set
  • Fields like Composer, BPM, Mood, Performer, and Disc Number are supported in the track editor
  • Custom / non-standard tags appear in the Custom Fields column and can be added or removed freely
  • Click Save to write changes to disk — changes are not auto-saved

Cover Art

  • Click the cover thumbnail on the album editor to open the artwork panel
  • Upload a file by clicking, dragging an image onto the panel, or pasting a URL
  • Use Search Covers to find artwork from iTunes, Deezer, MusicBrainz, and Bandcamp
  • Cover art is embedded in every FLAC file and also saved as cover.jpg in the album folder
  • Use Cover Clean-up in the sidebar to find albums with missing or small artwork
  • Sort Cover Clean-up by Name, Recently Added, or Size (small → large) to prioritize your work

MusicBrainz

  • Click MusicBrainz Lookup on the album editor to search for release metadata
  • Results include release date, label, country, track listing, and MusicBrainz IDs
  • Select the best matching release and click Apply to fill shared and per-track fields
  • MusicBrainz IDs are written as tags and help other tools (like Beets or Picard) identify releases
  • Searches use Album + Album Artist from the current editor values — fill those in first for best results

ReplayGain

  • Click ReplayGain on the album editor to calculate loudness normalization values
  • Uses ffmpeg's EBU R128 algorithm — produces album gain and per-track gain values
  • A preview shows the calculated gain before you choose to write it
  • Results are written as standard ReplayGain tags in each FLAC file
  • Requires ffmpeg to be installed (included in the Docker image automatically)

Lyrics

  • Click the lyrics icon (♪) on any track row to open the lyrics panel
  • Click Fetch Lyrics to search LRCLib for synchronized (timestamped) lyrics
  • Synced lyrics are saved as a .lrc sidecar file next to the FLAC
  • Plain lyrics are written into the LYRICS FLAC tag via the track editor
  • Track rows show a ♪ icon when lyrics are available (tag or .lrc file)

File Organizer

  • Click Organize on the album editor to preview a rename/move based on tags
  • The pattern uses tokens like {album_artist}, {album}, {track:02d}, {title}
  • A preview shows the exact source → destination paths before any files move
  • Confirm to move files; the library index updates immediately — no rescan needed
  • Set ORGANIZE_TARGET in your .env to enable this feature
  • Save frequently-used patterns for quick recall

Trash & Recovery

  • Click the trash icon on any track row or Move to Trash on the album editor to soft-delete
  • Files move to .trash/ inside your media root — nothing is permanently deleted
  • Open Trash in the sidebar to browse trashed items grouped by artist and album
  • Click Recover, Restore Album, or Restore Artist to move files back
  • Click Empty Trash to permanently purge all trashed files
{% endif %} {% if navidrome_enabled %}

Navidrome

  • Click Sync Navidrome on the album editor to pull play counts, ♥ favorites, and ★ ratings
  • Heart (♥) and star rating columns appear on each track row in the album editor
  • Changes to ♥ and ★ are written back to Navidrome in real time via the Subsonic API
  • Use the ♥ and ★ filters in the explorer sort bar to browse your library by Navidrome data
  • Click Trigger Navidrome Scan to ask Navidrome to re-index your library after tagging
{% endif %}

Themes & Settings

  • Click the ⚙ gear icon in the top-right to open Settings
  • Choose from six dark themes: Default, Nord, Dracula, GitHub Dark, Tokyo Night, Catppuccin
  • Adjust thumbnail size and covers-per-page for the Cover Clean-up view
  • Set your preferred cover filename (cover.jpg or folder.jpg)
  • Switch between multiple configured libraries using the library picker in the sidebar
  • The Library Scan button re-indexes your entire music folder when needed
